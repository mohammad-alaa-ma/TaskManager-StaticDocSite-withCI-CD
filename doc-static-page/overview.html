Overview

This project demonstrates deploying a 3-tier Task Management application and a Static Documentation Website on Kubernetes.

The database is MySQL running on an external machine, ensuring persistence and separation between application pods and data storage.

Technology Stack

The Task Manager application is built with modern web technologies:

  • Frontend: HTML, CSS, JavaScript
  • Backend: Node.js
  • Database: MySQL
  • Containerization: Docker
  • Orchestration: Kubernetes
  • GitOps & CI/CD: ArgoCD
  • Version Control: Git & GitHub
